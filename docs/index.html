Factorio ModPack Generator

Instructions (English)

This is a tool to easily generate a ModPack for Factorio. It packages the mods enabled in your latest save data into a zip file.

To generate the ModPack, you will need the `mod-list.json` file, which is located in the mods directory. If you are unsure where the mods directory is, please refer to `Location of mods directory`. Here are the specific steps for using the tool:

Enter your name or handle in the `Please enter your name (or handle)` field.
Open the `mod-list.json` file located in the mods directory, copy its contents, and paste them into the `Please enter your mod-list.json` field.
Press the `Process` button to download the zip file.
Move the downloaded zip file to the mods directory.
When you start Factorio, the created ModPack will be reflected. The name of the mod will be `modpack-by-{your name}`.
説明 (日本語)

これはFactorioのModPackを簡単に生成するツールです。最新のセーブデータで有効化されているMODをModPackとしてパッケージングしてzipファイルを作成します。

生成には`mod-list.json`が必要になりますが、これはmodsディレクトリの中にあります。modsディレクトリの場所がわからない方は`Location of mods directory`を参照してください。 具体的な使用方法は以下のとおりです。

`Please enter your name (or handle)`にあなたの名前もしくはハンドルネームを入力します
modsディレクトリの中にある`mod-list.json`を開いて内容をコピーし、`Please enter your mod-list.json`にペーストします。
`Process`のボタンを押すとzipファイルがダウンロードできます。
ダウンロードしたzipファイルをmodsディレクトリの中に移動します。
Factorioを起動すると作成したModPackが反映されます。Modの名前は`modpack-by-{あなたの名前}`となります。
Location of mods_directory
  • Windows: C:\Users\[Your Username]\AppData\Roaming\Factorio\mods
  • macOS: /Users/[Your Username]/Library/Application Support/factorio/mods
  • Linux: /home/[Your Username]/.factorio/mods
